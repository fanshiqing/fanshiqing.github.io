【闲情】记 第二届南大上海校友会新年羽毛球嘉年华

发表于 2017-01-07 | 分类于 闲情 | 0 comments | 浏览 次

Abstract

  • 时间： 2017年1月7日
  • 地点：上海财大（武川路校区）羽毛球馆（旧馆）
  • 事件：第二届南大上海校友会新年羽毛球嘉年华活动
  • 角色：第二次参加这个活动，本次担任 杨浦队 队长 ^_^
  • 赛制：趣味赛 + 团体赛

  • 赛果：杨浦队在8支参赛队中勇夺 第4名 (历史最好成绩)！

    • 趣味赛:
      1. 单手颠球，穿越障碍
      2. 三人隔网，你发我接（限时）
    • 团体赛(8支)
      • 徐汇、闵松、沿江、张江、花木、杨浦、长静黄联队、市北联队

  • 赛后聚餐地点：上海财大豪生酒店
